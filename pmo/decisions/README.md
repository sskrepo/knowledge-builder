# Decisions

Open decisions awaiting user input, plus a record of past decisions.

Filename convention: `DECISION-{NNN}-{kebab-slug}.md` (zero-padded 3-digit ID)

Format: see [dev-agent-team/shared/decision-protocol.md](../../../dev-agent-team/shared/decision-protocol.md).

## Numbering

Sequential, project-wide, never reused. TPM keeps the counter.

Current next ID: **001**

## Status legend

- `open` — awaiting user decision
- `decided` — user has chosen
- `superseded` — replaced by a later decision (link to it)
